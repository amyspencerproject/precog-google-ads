Custom Query Fields

You can build custom queries to refine the data you access from the Google Ads API. Navigate to the Google Ads Query Builder and select a resource. This will open a new query with the resource listed after FROM in the query string. Then choose SELECT to refine your Custom Query. Do not use the other types i.e. WHERE, ORDER BY, LIMIT, or PARAMETERS. Your query will be a string with the form SELECT ... FROM. For example, the resource ‘Ad Group’ (FROM ad_group) querying data from fields - id, name, campaign name, and clicks - will have the query string SELECT ad_group.id, ad_group.name, campaign.name, metrics.clicks FROM ad_group.
